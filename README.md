# 🏦 PIG Bank

Aplicação web desenvolvida com React que simula um sistema bancário digital, permitindo o gerenciamento de saldo, entradas e saídas financeiras de forma dinâmica e intuitiva.

O projeto foi criado com foco em simular regras de negócio reais, aplicando conceitos modernos de desenvolvimento front-end como componentização, gerenciamento de estado e construção de interfaces SPA (Single Page Application).

---

## 🚀 Tecnologias utilizadas

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3

---

## 🎯 Funcionalidades

* 💰 Cadastro de entradas e saídas financeiras
* 📊 Atualização dinâmica do saldo em tempo real
* 🔄 Interface dinâmica sem recarregamento de página (SPA)
* 🧩 Componentes reutilizáveis
* 📱 Layout responsivo (Mobile First)

---

## 🧠 Conceitos aplicados

* Gerenciamento de estado no React
* Componentização e reutilização de código
* Separação de responsabilidades
* Estruturação de projetos front-end escaláveis
* Boas práticas de desenvolvimento

---

## 💡 Objetivo do projeto

Simular o funcionamento de uma aplicação bancária real, focando na construção de interfaces funcionais e na manipulação de dados em tempo real, aproximando o desenvolvimento de cenários do mercado.

## 📂 Como executar o projeto

```bash
# Clonar repositório
git clone https://github.com/Luizfelipeosz/PIG---BANK

# Acessar pasta
cd PIG---BANK

# Instalar dependências
npm install

# Rodar projeto
npm start
```
